Redesigned document detail screen

The document detail screen has been completely redesigned with clearer metadata
and sheet-based editing for tags, correspondent, document type, storage path,
dates, ASN, custom fields, and owner. The PDF now pages horizontally with a page
indicator, and the full-screen preview adds in-document search and smoother
scrolling. You can delete documents from the detail screen, see how many notes a
document has, and include the original file when sharing.


Multi-column iPad layout

The document list and detail screen now use a full multi-column layout on iPad,
making much better use of the larger screen.


Nested tags

The app now supports Paperless-ngx nested (hierarchical) tags.


Other Features

- Quick add: create a tag, correspondent, or document type on the fly right from
  the picker
- Show your server's configured app title as a friendly name in the server
  picker and connection settings
- Deep links can now open a specific document field for editing
- Document cells in the list now show the number of notes a document has
- Add a loading indicator to the search bar while results are loading
- Renamed "Log out" to "Disconnect" to better reflect what the action does
- Improved UI responsiveness when loading large amounts of data
- Ready for saved view permissions and the new task API in the upcoming
  Paperless-ngx v3


Bug Fixes

- Show the actual error from your identity provider during OIDC login instead of
  a cryptic message
- Fix OIDC login failing behind some reverse proxies
- Fix client-certificate (mTLS) login
- Fix a freeze when polling tasks on installations with many pending tasks
- Fix the task view briefly showing "no elements" while still loading
- Hide the remove-inbox-tag action for documents that don't have inbox tags
